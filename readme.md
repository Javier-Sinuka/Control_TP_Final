## ⚙️ Tecnologías utilizadas

- Quarkdown (generación del informe)
- GitHub Actions (automatización CI/CD)
- GitHub Pages (visualización web)

---

## 🔄 Flujo de trabajo automático

Cada push a la rama `main`:

1. Se compila el archivo `.qd`
2. Se genera el HTML del informe
3. Se publica automáticamente en GitHub Pages

---

## 📌 Nota importante

El contenido del informe se genera automáticamente mediante GitHub Actions.

👉 No es necesario (ni recomendable) subir archivos generados como `/output`.

---

## 🌐 Sobre GitHub Pages

El sitio publicado utiliza como entrada un archivo `index.html`, generado automáticamente en cada build.
